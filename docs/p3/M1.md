# M1：定长 KV + 宿主 18 步 AR 环 + 视觉 tokenizer stage

M1 在 M0（GPT 主干 + KV-cache `prefill/decode_step`）基础上补齐两块：
1. **宿主侧 18 步自回归解码环**（`valid_kv_len` 逐步推进、KV buffer 就地更新）；
2. **视觉 tokenizer stage**（DINOv3/EVA02 视觉塔，独立 `embed_visual` engine）。

> 总体方案见 `docs/p3/arch.md`；M0 见 `docs/p3/M0.md`。

## 2. 视觉 tokenizer stage（`embed_visual`）

忠实复刻 `laser_model_export/.../encoder/dinov3_encoder.py::Dinov3Encoder`（`eva_cfg={}` → timm 全默认）：

```
frame[1,3,H,W]
  → patchify    Conv2d(3,64,k2,s2)
  → downsample  DownsampleBlock(64,256,k3,stride=(1,2))     # SiLU→conv1→SiLU→conv2 + shortcut
  → patch_embed Conv2d(256,384,k16,s16) → [1,Hp,Wp,384] 展平 token
  → [cls(1)+reg(4)] 前缀 + EvaBlock×12                       # dinov3 2D-RoPE / LayerScale / SwiGLU
  → 丢前缀 → reshape[1,384,Hp,Wp]
  → expand      Conv2d(384,1024,k3,s1,p1)
  → flatten     → [1, Hp*Wp, 1024]
```

**EvaBlock**（`init_values=1e-5` → LayerScale）：
`x = x + w1·attn(LN1(x)); x = x + w2·SwiGLU(LN2(x))`。
- `EvaAttention`：`qkv` Linear(bias=False) → 6 头；dinov3 2D-RoPE 只加在 patch token（前缀 5 个 cls/reg 不加）；`scale_norm=False`（无内层 norm）；`proj` Linear(bias=True)。
- `SwiGLU`：`fc2(SiLU(fc1_g(x)) * fc1_x(x))`，hidden=1536。

**dinov3 2D-RoPE**：按 patch 网格 `(Hp,Wp)` 在**宿主 numpy 预算** sin/cos 并 bake 为常量（对齐 `pos_embed_sincos.RotaryEmbeddingDinoV3`：0.5-centered、separate 归一、[-1,1]、`periods=100^(2·arange(16)/32)`、`rotate_half` repeat 拼接）；图内只做 `x·cos + rot_half(x)·sin`。这正是 arch.md 的「2D RoPE 位置表按 patch 网格预算，cos/sin 传入/bake」。

**边界（M1 范围）**：本 stage 只做**单帧 DINOv3 backbone**。多视角/时序拼接、`view_token` 插入、PE 投影、`hist_img_feat` 时序缓存等编排（数据无关的静态 Python 循环）留在宿主侧（`VisualTokenizer.forward` 的外层逻辑），后续 milestone 再串成完整 encode。数据相关 crop/resize 走宿主预处理（PVA/hipp），engine 只吃已 crop 的定尺寸帧。

## 3. 宿主 AR 环（`Plannn3Runner`）

对齐 `network.planner_generate` + mlc-vla `PiZeroRunner`：

```text
logits0, kv = prefill(token_embeds)          # 首 token（begin）→ 首 logits
id0 = argmax(logits0)
for step in range(pred_times-1):
    pos = prompt_len + step                  # 写入位置 / RoPE 位置 / 有效长度
    emb = embed_token(id_prev)               # [1,1,n_embd]
    step_cos/step_sin = LLM interleaved rope(offset=pos)
    add_mask   = (idx<=pos ? 0 : -inf)        # [1,1,1,max_seq]
    write_onehot = (idx==pos)                 # [1,max_seq,1]
    logits, kv = decode_step(emb, step_cos, step_sin, add_mask, write_onehot, kv)
    id = argmax(logits)
```

`valid_kv_len`/写入位置/RoPE 位置全部在宿主推进并以 tensor 交给图（trace 稳定），KV buffer 每步由图返回新句柄、宿主回传，无 concat、shape 恒定。argmax 放在宿主（M2 可整图化 `decode_loop_kv` + CUDA Graph）。

## 4. 如何运行

```bash
export TVM_HOME=/path/to/tvm
export TVM_LIBRARY_PATH=$TVM_HOME/build
export PYTHONPATH=$TVM_HOME/python:$(pwd)/python:$PYTHONPATH

# 视觉塔（dummy 小图秒级）
python -m mlc_vla.plannn3_compile --dummy --visual --target c
python -m mlc_vla.plannn3_compile --dummy --visual --dump-ir      # 看视觉塔 IR

# 宿主 18 步 AR 环
python -m mlc_vla.plannn3_compile --dummy --generate --target c

# 全尺寸视觉塔（front main 512×1536，c 后端较慢）
python -m mlc_vla.plannn3_compile --visual --target c
```
