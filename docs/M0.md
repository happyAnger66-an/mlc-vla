# M0：π0.5 单步前向在 TVM 跑通

M0 是 `mlc-vla` 的第一步，目标：用 TVM `relax.frontend.nn` 忠实实现 π0.5 的单步前向
（SigLIP 视觉 + prefix embedding + 双专家联合注意力 + 单步 flow-matching 去噪），
并提供与 openpi 的数值对拍脚手架。

> 架构总体设计见 `docs/arch.md`。本文件只覆盖 M0 的落地细节与现状。

## 1. 已实现

| 模块 | 文件 | 说明 |
|------|------|------|
| 配置 | `model/pi0/pi0_config.py` | 双专家 Gemma（gemma_2b + gemma_300m）、SigLIP So400m/14、动作维度，对齐 openpi |
| 双专家主干 | `model/pi0/gemma_dual.py` | 联合注意力（各专家独立 QKV、沿序列拼接）、RoPE（半分旋转）、Gemma 门控 FFN、普通/adaRMS、gated residual |
| 视觉塔 | `model/pi0/siglip.py` | SigLIP 27 层 + multi-modal projector，patch-embedding 用「展平 patch + Linear」等价实现 |
| 顶层模型 | `model/pi0/pi0_model.py` | `embed_image` / `embed_language` / `denoise_step` 三个可导出函数 + mask/RoPE 常量 bake |
| 权重 loader | `model/pi0/pi0_loader.py` | openpi→MLC 映射**骨架**（含 6 条转换规则，键名待对照 checkpoint 补全） |
| 编译/冒烟 | `compile.py` | 导出 IRModule、最小 relax pipeline、随机权重 `denoise_step` 冒烟 |
| 对拍 | `compare.py` | 与 openpi 单步 v_t 的 cosine 对拍**脚手架** |

## 2. 关键设计取舍（M0 优先正确性）

- **不用 PagedKVCache**：用显式拼接 + 加性 mask 的 eager 注意力，逐算子对齐 openpi
  eager 路径；CUDA Graph / KV cache 留到 M1/M2。
- **固定 shape**：batch=1，prefix/suffix 长度编译期已知，故 RoPE 的 cos/sin 表与注意力
  mask 直接作为常量 bake 进图。
- **时间编码在宿主侧预算**：正弦时间编码用 numpy 算好后作为 `time_emb` 传入，in-graph
  只跑 time MLP（产生 adaRMS 条件），避免依赖图内 sin/cos。
- **dtype=float32**：便于 CPU 对拍；bf16/混精在后续 phase 切换。
- **π0.5 细节**：suffix 只含动作 token（state 进离散语言 token）；expert-0(PaliGemma)
  用普通 RMSNorm，expert-1(action) 用 adaRMS；`(1+scale)` 在 forward 内做，权重直接搬。

## 3. 数值对齐基准（与 openpi 的对照点）

- 配置：`vlm=(2048/18/16384/8heads/1kv/256hd)`，`action=(1024/18/4096/8/1/256)`，
  `num_patches=256`，3 路图像，`prefix_len=968`，`suffix_len=50`（已脚本校验）。
- RoPE：`freq=(2/H)*arange(H/2)`，`theta=1e4`，半分旋转（对齐 `gemma._apply_rope`）。
- 注意力：q 先 `*head_dim^-0.5`，logits float32，big_neg=`-2.3819763e38`，GQA(K=1) 广播。
- mask：prefix 只看 prefix；suffix 看 prefix+全部 suffix（对齐 `make_att_2d_masks`）。

## 4. 如何运行

前置：需要可用的 TVM（Unity/Relax，含 `tvm.relax.frontend.nn`）。
本仓 `3rdparty/tvm` 或 `edgeLLM/tvm` 构建后，将其 `python` 加入 `PYTHONPATH`。

```bash
export PYTHONPATH=/path/to/tvm/python:$(pwd)/python:$PYTHONPATH

# 1) 打印导出的 relax IRModule
python -m mlc_vla.compile --dummy --dump-ir

# 2) 冒烟：随机权重跑通 denoise_step（CPU）
python -m mlc_vla.compile --dummy --smoke --target llvm
python -m mlc_vla.compile --smoke --target llvm   # 全尺寸

# 3) 与 openpi 单步对拍（需装 openpi + 补全 loader）
python -m mlc_vla.compare --mode A
```

纯 Python 部分（配置、mask、loader 变换）不依赖 TVM，已通过校验。
